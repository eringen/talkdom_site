post-json:apply:json:

JSON Method

Description

Sends a JSON payload with a POST request. The JSON string is parsed at runtime and stringified as the request body. The response is applied to the receiver.

Syntax

receiver post-json: <url> apply: <op> json: <json-string>

Example

<div receiver="result"></div>

<button sender="result post-json: /api/data apply: inner json: {&quot;name&quot;:&quot;test&quot;}">
  Send JSON
</button>

Notes

  • Use HTML entities (&quot;) for quotes inside the attribute
  • Invalid JSON silently falls back to an empty object {}
  • CSRF token is sent automatically for POST requests
  • See also: put-json:apply:json:
