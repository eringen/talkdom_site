delete:

HTTP Method

Description

Performs an HTTP DELETE request to the specified URL. Returns the response body as text. Automatically includes a CSRF token from the page's meta tag.

Syntax

receiver delete: <url>

Example

<div receiver="item" id="item-42">
  Item content here
  <button sender="item confirm: Delete this item? | item delete: /api/items/42 apply: outer">
    Delete
  </button>
</div>

Notes

  • Returns a Promise resolving to response text
  • Automatically includes X-CSRF-Token header from <meta name="csrf-token">
  • Sends all standard talkDOM request headers
  • Use delete:apply: as a convenient shorthand
  • Commonly paired with confirm: for destructive operations
