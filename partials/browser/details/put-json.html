put-json:apply:json:

JSON Method

Description

Sends a JSON payload with a PUT request. The JSON string is parsed at runtime and stringified as the request body. The response is applied to the receiver.

Syntax

receiver put-json: <url> apply: <op> json: <json-string>

Example

<div receiver="result"></div>

<button sender="result put-json: /api/posts/1 apply: inner json: {&quot;title&quot;:&quot;Updated&quot;}">
  Update
</button>

Notes

  • Identical to post-json: but uses the PUT method
  • CSRF token is sent automatically
