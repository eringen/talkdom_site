X-TalkDOM-Current-URL

Request Header

Description

Contains the current page URL (location.href) when the request was made. Useful for server-side context such as highlighting the active nav item.

Header

X-TalkDOM-Current-URL: https://example.com/docs

Notes

  • Sent with all HTTP methods
  • Reflects the URL at the time of the request, including any push-url changes
  • Useful for breadcrumbs, active states, or contextual server responses
