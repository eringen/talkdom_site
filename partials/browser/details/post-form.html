post-form:apply:form:

Form Method

Description

Serializes a form's inputs as application/x-www-form-urlencoded and sends them with a POST request. The response is applied to the receiver.

Syntax

receiver post-form: <url> apply: <op> form: <selector>

Example

<form id="my-form">
  <input name="name" value="test">
  <input name="email" value="user@example.com">
</form>

<div receiver="result"></div>

<button sender="result post-form: /api/submit apply: inner form: #my-form">
  Submit
</button>

Notes

  • Uses URLSearchParams and FormData for serialization
  • The form: argument is a CSS selector (e.g. #my-form)
  • CSRF token is sent automatically for POST requests
  • See also: put-form:apply:form:, delete-form:apply:form:
