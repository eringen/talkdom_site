talkDOM.send()

Programmatic API

Description

Dispatch messages programmatically from JavaScript. Parses the same message syntax as sender attributes, including pipes and semicolons.

Signature

talkDOM.send(raw: string): Promise

Examples

// Simple message
await talkDOM.send("content get:apply: /page.html inner");

// Pipe chain
await talkDOM.send("content get: /data | content apply: append");

// Parallel chains
await talkDOM.send("a get:apply: /x inner; b get:apply: /y inner");

// Trigger on page load
document.addEventListener("DOMContentLoaded", function() {
  talkDOM.send("feed get:apply: /api/latest inner");
});

Notes

  • Returns a Promise that resolves when all chains complete
  • Useful for triggering messages from custom event handlers, timers, or other JS code
  • Does not push URL to history — use history.pushState() manually if needed
