receiver

Attribute

Description

Marks an element as a message receiver. The value is the receiver's name, used to address it in sender messages. Multiple elements can share the same name — messages are delivered to all matching receivers.

Syntax

<div receiver="name"></div>

<!-- With polling -->
<div receiver="name get:apply: /url inner poll: 5s"></div>

Example

<div receiver="content">Initial content</div>

<button sender="content get:apply: /page.html inner">
  Load
</button>

Notes

  • The name is the first word of the attribute value
  • Additional words define polling behavior (method + interval)
  • Receivers are found via [receiver~="name"] CSS selector
  • Can be combined with accepts and persist attributes
