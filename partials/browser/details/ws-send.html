ws:send:

WebSocket

Description

Sends the receiver element's value (for inputs/textareas/selects) or text content over an existing WebSocket connection.

Syntax

receiver ws:send: <ws-url>

Example

<input receiver="chatbox" type="text">

<button sender="chatbox ws:send: ws://localhost:3000/chat">
  Send
</button>

Notes

  • The connection must already be open (established via ws: on a receiver)
  • For input/textarea/select elements, sends el.value
  • For other elements, sends el.textContent
  • Rejects with "not connected" if no open connection exists for the URL
