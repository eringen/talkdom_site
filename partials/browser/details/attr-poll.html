poll:

Attribute

Description

Automatically repeats a message at a fixed interval. Added to the receiver attribute after the method keywords. Polling stops automatically when the element is removed from the DOM.

Syntax

<div receiver="name method: args poll: <interval>"></div>

Interval Formats

  • 5s — every 5 seconds
  • 500ms — every 500 milliseconds
  • 30s — every 30 seconds

Example

<!-- Auto-refresh feed every 10 seconds -->
<div receiver="feed get:apply: /api/feed inner poll: 10s">
  Loading...
</div>

Notes

  • poll: must be the last keyword in the receiver attribute
  • Uses setInterval — cleared when the element is disconnected
  • The method and args before poll: are executed on each tick
  • Dynamically added elements with poll: are detected via MutationObserver
  • Default limit of 64 concurrent pollers (configurable via talkDOM.maxPollers)
