put-form:apply:form:

Form Method

Description

Serializes a form's inputs as application/x-www-form-urlencoded and sends them with a PUT request. The response is applied to the receiver.

Syntax

receiver put-form: <url> apply: <op> form: <selector>

Example

<form id="edit-form">
  <input name="title" value="Updated title">
</form>

<div receiver="result"></div>

<button sender="result put-form: /api/posts/1 apply: inner form: #edit-form">
  Update
</button>

Notes

  • Identical to post-form: but uses the PUT method
  • CSRF token is sent automatically
