talkDOM.ws

WebSocket API

Description

Programmatic API for managing WebSocket connections. Available when the talkdom-ws.min.js plugin is loaded.

Methods

talkDOM.ws.connect(url)

Open a WebSocket connection to the given URL.

talkDOM.ws.send(url, data)

Send data over an open connection. Returns true if sent, false if not connected. Objects are JSON-stringified.

talkDOM.ws.disconnect(url)

Close the connection and clean up resources for the given URL.

Properties

talkDOM.ws.connections

Read-only object mapping URLs to { state, receivers }.

talkDOM.ws.maxConnections

Maximum concurrent connections. Default: 16.

Example

// Connect programmatically
talkDOM.ws.connect("ws://localhost:3000/live");

// Send structured data
talkDOM.ws.send("ws://localhost:3000/live", {
  action: "subscribe",
  channel: "news"
});

// Check connection status
console.log(talkDOM.ws.connections);

// Disconnect
talkDOM.ws.disconnect("ws://localhost:3000/live");
