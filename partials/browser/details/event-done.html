talkdom:done

Event

Description

Fires on the receiver element when a message completes successfully. The event bubbles up the DOM tree, so you can listen on any ancestor or on document.

Event Detail

{
  receiver: "content",    // receiver name
  selector: "get:apply:", // method selector
  args: ["/url", "inner"] // parsed arguments
}

Example

document.addEventListener("talkdom:done", function(e) {
  console.log("Completed:", e.detail.selector);
  console.log("Receiver:", e.detail.receiver);
});

Notes

  • Fires after the DOM has been updated (for apply operations)
  • For outer swaps, fires on the replacement element or parent
  • Dispatched as a CustomEvent with bubbles: true
