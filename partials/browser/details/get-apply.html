get:apply:

Combined Method

Description

Performs a GET request and applies the response to the receiver in one step. Shorthand for get: piped to apply:.

Syntax

receiver get:apply: <url> <operation>

Example

<div receiver="content"></div>

<button sender="content get:apply: /page.html inner">
  Load Page
</button>

Operations

  • inner — replace innerHTML
  • text — set textContent
  • append — add content at the end
  • outer — replace entire element

Notes

  • Equivalent to receiver get: /url | receiver apply: inner
  • Respects the accepts attribute on the receiver
  • Triggers persist if the attribute is present
