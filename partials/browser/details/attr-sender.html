sender

Attribute

Description

Makes an element dispatch talkDOM messages on click. The attribute value is the full message string, following Smalltalk keyword syntax.

Syntax

<button sender="receiver method: arg">Click me</button>

<!-- With pipes -->
<button sender="content get: /url | content apply: inner">Load</button>

<!-- With parallel chains -->
<button sender="a get:apply: /x inner; b get:apply: /y inner">Both</button>

Example

<a href="/docs"
   sender="content get:apply: /partials/docs.html inner"
   push-url="/docs">
  Docs
</a>

Notes

  • Works on any element, not just buttons or links
  • Click events are delegated — uses closest("[sender]")
  • Prevents default click behavior (e.g., link navigation)
  • Combine with push-url for SPA-style navigation
