put:

HTTP Method

Description

Performs an HTTP PUT request to the specified URL. Returns the response body as text. Automatically includes a CSRF token from the page's meta tag.

Syntax

receiver put: <url>

Example

<div receiver="item"></div>

<button sender="item put: /api/items/42 | item apply: inner">
  Update Item
</button>

Notes

  • Returns a Promise resolving to response text
  • Automatically includes X-CSRF-Token header from <meta name="csrf-token">
  • Sends all standard talkDOM request headers
  • Use put:apply: as a convenient shorthand
