post:

HTTP Method

Description

Performs an HTTP POST request to the specified URL. Returns the response body as text. Automatically includes a CSRF token from the page's meta tag.

Syntax

receiver post: <url>

Example

<meta name="csrf-token" content="abc123">

<div receiver="result"></div>

<button sender="result post: /api/items | result apply: inner">
  Create Item
</button>

Notes

  • Returns a Promise resolving to response text
  • Automatically includes X-CSRF-Token header from <meta name="csrf-token">
  • Sends all standard talkDOM request headers
  • Use post:apply: as a convenient shorthand
