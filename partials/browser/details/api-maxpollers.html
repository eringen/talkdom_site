talkDOM.maxPollers

API

Description

Gets or sets the maximum number of concurrent polling intervals. Prevents unbounded setInterval creation on pages with many polling receivers.

Default

64

Example

// Increase the limit
talkDOM.maxPollers = 128;

// Check current limit
console.log(talkDOM.maxPollers); // 128

Notes

  • Once the limit is reached, new poll: receivers are ignored with a console warning
  • Pollers clean up automatically when their element is removed from the DOM
  • Getter/setter property — not a method
