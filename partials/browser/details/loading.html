loading:

Control Method

Description

Toggles a CSS class on the receiver element while an async operation is in progress. The class is added before the request starts and removed when it completes (on both success and error).

Syntax

receiver method: <args> loading: <class-name>

Example

<style>
  .is-loading { opacity: 0.5; pointer-events: none; }
</style>

<div receiver="content"></div>

<button sender="content get: /api/data apply: inner loading: is-loading">
  Load
</button>

Notes

  • Can be combined with any async method (get:, post:, etc.)
  • The loading: keyword is stripped before method lookup
  • Class is removed on both success and error — no stuck loading states
  • Works with Tailwind utility classes (e.g. opacity-50)
