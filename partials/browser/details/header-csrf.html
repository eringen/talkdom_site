X-CSRF-Token

Request Header

Description

CSRF token automatically extracted from a <meta> tag and sent with POST, PUT, and DELETE requests. Not sent with GET requests.

Setup

<meta name="csrf-token" content="your-token-here">

Header Sent

X-CSRF-Token: your-token-here

Notes

  • Only sent with mutating methods: post:, put:, delete:
  • Not sent with get: requests
  • If no <meta name="csrf-token"> is found, the header is omitted
  • Validate this header server-side to prevent cross-site request forgery
