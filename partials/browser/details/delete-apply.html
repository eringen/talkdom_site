delete:apply:

Combined Method

Description

Performs a DELETE request and applies the response to the receiver in one step. Includes CSRF token automatically.

Syntax

receiver delete:apply: <url> <operation>

Example

<div receiver="item" id="item-1">
  Item content here
</div>

<button sender="item delete:apply: /api/item/1 outer">
  Delete
</button>

Notes

  • Sends X-CSRF-Token from <meta name="csrf-token">
  • Use with outer to remove the element entirely after deletion
  • Combine with confirm: for safety
