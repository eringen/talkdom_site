talkdom:error

Event

Description

Fires on the receiver element when a message fails — due to an HTTP error, network failure, or a rejected Promise (e.g., from confirm:). Bubbles up the DOM.

Event Detail

{
  receiver: "content",
  selector: "get:apply:",
  args: ["/url", "inner"],
  error: 404             // HTTP status or error object
}

Example

document.addEventListener("talkdom:error", function(e) {
  console.error("Failed:", e.detail.selector, e.detail.error);
  // Show user-friendly error message
  var el = document.querySelector('[receiver="' + e.detail.receiver + '"]');
  if (el) el.innerHTML = '<p class="text-red-400">Failed to load.</p>';
});

Notes

  • The error field contains the HTTP status code or a network error object
  • For confirm: cancellation, error is "cancelled"
  • Useful for showing fallback UI or retry mechanisms
