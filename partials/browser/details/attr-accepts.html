accepts

Attribute

Description

Restricts which apply operations a receiver allows. The value is a space-separated list of permitted operations. If omitted, all operations are allowed.

Syntax

<div receiver="name" accepts="inner append"></div>

Valid Values

  • inner — allow innerHTML replacement
  • text — allow textContent setting
  • append — allow appending content
  • outer — allow outerHTML replacement

Notes

  • Disallowed operations log a console error and are silently skipped
  • Useful for safety — prevent accidental outer swaps that could remove elements
