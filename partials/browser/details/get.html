get:

HTTP Method

Description

Performs an HTTP GET request to the specified URL. Returns the response body as text, which can be piped to subsequent messages.

Syntax

receiver get: <url>

Example

<div receiver="content"></div>

<button sender="content get: /page.html | content apply: inner">
  Load Page
</button>

Notes

  • Returns a Promise resolving to response text
  • Pipe the result to apply: to update the DOM
  • Use get:apply: as a convenient shorthand
  • Sends X-TalkDOM-Request, X-TalkDOM-Current-URL, and X-TalkDOM-Receiver headers
