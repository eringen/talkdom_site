persist

Attribute

Description

Saves receiver content to localStorage after every apply operation. Content is automatically restored on page load.

Syntax

<div receiver="sidebar" persist></div>

Example

<!-- User preferences persist across page reloads -->
<div receiver="theme" persist>light</div>

<button sender="theme apply: dark text">Dark Mode</button>
<button sender="theme apply: light text">Light Mode</button>

Notes

  • Boolean attribute — no value needed
  • Stored under localStorage key "talkDOM:<name>"
  • Stores both the content and the operation type (inner or outer)
  • Restored automatically via innerHTML or outerHTML on page load
