X-TalkDOM-Request

Request Header

Description

Sent with every talkDOM HTTP request. Value is always "true". Use this server-side to detect talkDOM requests and return partial HTML instead of full pages.

Header

X-TalkDOM-Request: true

Server-Side Example

// Express.js
app.get("/page", (req, res) => {
  if (req.headers["x-talkdom-request"]) {
    res.send(partialHTML);  // Just the content
  } else {
    res.send(fullPageHTML); // Full page with shell
  }
});

Notes

  • Sent with all HTTP methods (GET, POST, PUT, DELETE)
  • Enables progressive enhancement — same URL serves full pages or partials
