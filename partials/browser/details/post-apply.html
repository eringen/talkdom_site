post:apply:

Combined Method

Description

Performs a POST request and applies the response to the receiver in one step. Includes CSRF token automatically.

Syntax

receiver post:apply: <url> <operation>

Example

<div receiver="result"></div>

<button sender="result post:apply: /api/submit inner">
  Submit
</button>

Notes

  • Sends X-CSRF-Token from <meta name="csrf-token">
  • Operations: inner, text, append, outer
  • Respects the accepts attribute on the receiver
