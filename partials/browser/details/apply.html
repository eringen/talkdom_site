apply:

DOM Method

Description

Applies content to the receiver element using the specified operation. Typically receives piped content from an HTTP method.

Syntax

receiver apply: <content> <operation>

Operations

inner

Sets innerHTML. Replaces all child content.

text

Sets textContent. Escapes HTML, text only.

append

Uses insertAdjacentHTML("beforeend"). Adds without removing existing content.

outer

Replaces the entire element via outerHTML. The receiver is removed from the DOM.

Example

<!-- Pipe from get: to apply: -->
<button sender="content get: /page.html | content apply: inner">
  Load
</button>

<!-- Or use the combined shorthand -->
<button sender="content get:apply: /page.html inner">
  Load
</button>

Notes

  • Respects the accepts attribute — disallowed ops are rejected with a console error
  • Triggers persist to save content to localStorage if the attribute is present
  • When piped, the content argument comes from the previous step's return value
